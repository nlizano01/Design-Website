CODING BASICS

What is Code?


Coding is used throughout our everyday lives. It is used to program the operating system which you are currently using along with the web browser you use to visit this site. In fact. This website which you are reading right now was created using code. This skill may also allow for various high-paying job opportunities and is certainly a useful one to have. In this website, you will learn everything there is to know about code in website development.


How will this site help?


This site will primarily focus on website development and will include information about HTML, CSS and Javascript, the 3 main languages used to code websites. HTML, or Hypertext Markup Language is used to create the structure for any websites. CSS or Cascading Style Sheets is used to style websites. Lastly, Javascript is used to add functionality. Our goal is to help you develop your coding skills regardless of your skill level.
