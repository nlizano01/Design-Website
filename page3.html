Important list of CSS properties:


color: Changes the text color (ex: blue)
background-color: Changes the background color (ex: orange)
font-family: Changes the font family (ex: arial)
background-image: url('add url here') Adds a background image
font-size: Changes the font size (use 'px'; ex: 15px)
width: Changes the width of an element (use 'px'; ex: 20px)
height: Changes the height of an element (use 'px' ex: 35px)
margin: Adds a margin to an element *More info later
border: Adds a border (ex: 2px solid black)
padding: Adds padding to an element *More info later
border-radius: Rounds a border (use 'px' ex: 15px)
position: Changes positioning (ex: fixed, absolute, etc.)
display: Changes the way an element is displayed (ex: grid, flex)
float: Changes the alignment of an element (ex: right, left)


Units of measurement


In CSS, there are multiple units of measurement which could be used. In the table above, you may have seen 'px' or pixels being used as an example. This is because that is the most commonly used unit of measurement in CSS. How ever, there are others such as 'em', 'rem' or 'vh' which could also be used to size objetcs.



The box model


In CSS, there is something known as the box model which is used to represent various elements. The box model includes 4 parts: the content, padding, border and margin. As an example, lets use a basic div *Info in 'Learn HTML'. The content would represent any content inside the div such as text, images, etc. The padding, however, is space that is not considered part of the content but is used to make an element bigger. Then, the border is as it sounds and represents the border of an element. Lastly, the margin is space outside the border that acts similar to padding with the main difference being that it is outside of the border. All of these sections could be customized with CSS *Note: to change the size of the content use 'width' and 'height'.




Positioning


One of the most important concepts to understand in CSS is positioning. This is what allows a coder to position elements the way in which they want to. As shown above in the list of important properties, you can choose the way in which you want an element to be positioned with the 'position:' property. There are multiple different positions that could be used such as 'relative', which positions an element relative to others and 'fixed', which places an element in a fixed area in the screen, though the most important one to understand is 'absolute'. This makes it so that an element always remains in the same area, however, unlike a 'fixed' element, an 'absolute' element moves when you scroll down. Once you set up your position, you may use the 'top', 'bottom', 'left' and 'right' properties to customize the positioning of an element. For example, if an element is set to 'position: absolute' and 'top: 40px', this means that that element is 40 pixels from the top of the webpage. Keep in mind that with 'position: absolute', this will change if a user scrolls down.
