What is the purpose of HTML?


HTML is essentially used as the basis of nearly all websites. In other words, it is essential for the majority of websites as it forms the structure from which they are built upon. For example, HTML allows you to add all the basic elements of a website (ex: images, links, paragraphs, titles, etc.). Additionally, HTML also contains certain parts such as links to other pages (ex: .css pages, .js pages) along with other important data. However, HTML on its own is not sufficient to create a website, which is where other languages such as CSS and Javascript are necessary. Regardless, we will teach you everything there is to know about HTML.



How to start an HTML document


The image below shows how to create the structure for an HTML document.



The < !DOCTYPE html> is used to declare that the document uses the HTML language. Then, the < html> tag is used to contain all the HTML elements within the code. The < head> tag is used to contain any code that is NOT visible within the actual document. For example, this contains the < title> tag which is used to define the title of the document, meta data and links to other files. However, the code contained within the < body> tag IS visible within the document.


Within the body of the document you could add various tags used for all sorts of different purposes. Keep in mind that in HTML, each element has an opening tag and a closing tag.



List of important HTML elements:


< p>< /p> Creates a paragraph
< h1>< /h1> Creates a large heading
< a href="put link here">< /a> Creates a link
< img src="put image address here"> Creates an image
< b>< /b> Bolds text
< i>< /i> Creates italic text
< mark>< /mark> Highlights text
< br>< /br> Creates a line-breaker (space)
< hr>< /hr> Creates a horizontal line
< center>< /center> Centers text
< button>< /button> Creates a button
< input>< /input> Creates an input box (ex: text, etc.)
< main>< /main> Used for the main content of a webpage
< div>< /div> An empty division to be customized with CSS
< !--Put comment here--> Creates a comment in HTML


How to create an HTML table


Tables are often an essential part of websites. Here is how to create them using HTML.
*NOTE you will have to also customize borders and background colors for the table to appear.



List of important HTML elements for tables:


< table>< /table> Creates a table
< thead>< /thead> Creates a table header
< tbody>< /table> Creates a table body
< tfoot>< /tfoot> Creates a table footer
< tr>< /tr> Creates a table row
< td>< /td> Creates a table cell
