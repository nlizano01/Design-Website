What is the purpose of Javacript?


While HTML is used for the basis of all websites and CSS is used for the styling of websites, Javascript is the language coders use for the functionality of websites. While HTML and CSS are considered coding languages, Javascript is considered a programming language. In this course, you will learn all about Javascript and how it works.

Variables


Vairbles are the basis of all programming languages. Basically, variables are things that vary within your code. In Javascript, there are three types of variables. The first type is the 'const' variable meaning constant. This is a type of variable that cannot be redefined (changed) within your code. The other two types of variables 'let' and 'var' can be redefined within your code. The difference between these two is that 'let' is only available in a specific block of code while 'var' can be redefined at any area within your code.


Data Types


The second most important concept to understand is data types. Essentially, data types refer to the type of information stored within a variable. In Javascript, there are eight data types: String, Boolean, Number, Bigint, Null, Undefined, Symbol and Object. In this course, we will be focusing on numbers, booleans and strings, though we will also go over objects later on.


Javascript Data Types:


String Refers to words and phrases. Must be wrapped in single quotation marks when used ('').
Number Refers to numbers (ex: 1, 2, 3, etc.)
Boolean Used for 'true' or 'false' statements. *NOTE: 'true' and 'false' are the only acceptable keywords

"=" vs "==" vs "==="


In earlier sections, you may have noticed multiple different types of equal signs being used. In fact, in Javascript, there are three types of equal signs and it is important for a programmer to know the difference between each one of them. The first type of equal sign you may have seen is the single equal sign. This is the most common type of equal sign and is generally used for declaring variables. Then, there is the double equal sign. This is used in cases such as if statements to see if values match. This, however, does not take into account data types. Lastly, the triple equal sign acts just as the double equal sign though this does take into account data types.



DOM Manipulations


The next topic that will be discussed in this Javascript course is DOM (Document object model) manipulations. These essentially allow programmers to modify the CSS of a webpage through Javascript. The image below is an example of that (it changes the text color of the element myDiv to cyan). In order to use DOM manipulations, add .style to an HTML element referenced in your Javascript document then set the property that you wish to change. Keep in mind that in Javascript, the syntax is different. This is why the colon, which would be used in CSS, is changed to an equal sign and why the value of your property must always be wrapped in single or double quotation marks.



Examples of DOM Manipulation Properties


element.style.backgroundColor = 'color'
element.style.width = '14px'
element.style.fontSize = '25px'
